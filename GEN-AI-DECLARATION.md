# Generative AI Content Declaration

This module, **Lancer NPC Creator**, has been developed with the assistance of Generative AI tools in compliance with standard software development practices and platform guidelines (such as the Foundry VTT AI Content Policy).

## Scope of AI Assistance

### 1. Software Code
- The codebase, including the user interface implementation (using Foundry VTT's modern `ApplicationV2` framework) and logic for scaling stats and features, was developed with pair-programming assistance from **Antigravity (Google's AI coding assistant)**.
- All code has been fully reviewed, refactored, tested, and is maintained by the human package author.

### 2. Localization & Translations
- The initial translation keys and values in English (`lang/en.json`) and Portuguese (`lang/pt-BR.json`) were generated/translated with AI assistance.
- All generated translations were reviewed and adapted to align with the core mechanics and terminology of the **Lancer RPG** game system.

## Compliance & Maintenance
- No user-facing, pre-packaged lore, narrative, or artistic assets (prepared content) in this repository were generated by artificial intelligence.
- The module author understands the entirety of the codebase and is fully prepared to maintain, debug, and update it for future versions of Foundry VTT.
